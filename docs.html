Chat Generation

POST /v1/chat/generate

Generate AI responses using various LLM providers (Google, OpenAI, Anthropic, Ollama, DeepSeek, OpenRouter).

Parameters:

Parameter Type Required Description
prompt string Yes The user prompt/question
provider string No LLM provider (default: "google")
model string No Specific model name
temperature number No Response randomness (0.0-2.0, default: 0.4)
system_message string No System message to guide AI behavior

Example Request:

curl -X POST "http://localhost:5454/v1/chat/generate" \ -H "Content-Type: application/json" \ -d '{ "prompt": "Explain quantum computing in simple terms", "provider": "openai", "model": "gpt-4", "temperature": 0.7 }'

GitHub Repository Analysis

POST /v1/github/answer

Analyze GitHub repositories and answer questions about codebases using AI.

Parameters:

Parameter Type Required Description
question string Yes Question about the repository
text string No Relevant file content or context
tree string No Repository file tree structure
summary string No Repository summary
chat_history string No Previous conversation context

Example Request:

curl -X POST "http://localhost:5454/v1/github/answer" \ -H "Content-Type: application/json" \ -d '{ "question": "How does authentication work in this codebase?", "text": "// auth.js content here...", "tree": "src/\n auth/\n auth.js\n middleware.js", "summary": "A Node.js authentication service" }'

Website Processing Tools

POST /v1/website/markdown

Convert any website to clean, readable markdown format.

Example Request:

curl -X POST "http://localhost:5454/v1/website/markdown" \ -H "Content-Type: application/json" \ -d '{"url": "https://example.com"}'
POST /v1/website/html-to-md

Convert raw HTML content to clean markdown format.

Example Request:

curl -X POST "http://localhost:5454/v1/website/html-to-md" \ -H "Content-Type: application/json" \ -d '{ "html": "

Title

Content with bold text.

" }'
